Kebijakan Privasi

Sistem Manajemen Tugas & Jadwal Kuliah Mahasiswa UNPER

Informasi Pribadi yang Kami Kumpulkan

Kami mengumpulkan informasi pribadi yang Anda berikan secara langsung saat mendaftar dan menggunakan layanan kami, termasuk:

  • Nama lengkap
  • Alamat email
  • Nomor Induk Mahasiswa (NIM)
  • Nama fakultas dan jurusan
  • Informasi akademik terkait tugas dan jadwal kuliah

Perlindungan Data Pribadi

Kami sangat menjaga kerahasiaan dan keamanan data pribadi Anda:

  • Password Anda disimpan dalam bentuk terenkripsi
  • Informasi login dilindungi dengan sistem otentikasi JWT
  • Kami tidak akan membagikan data pribadi Anda kepada pihak ketiga
  • Anda dapat menghapus akun Anda kapan saja dengan menghubungi administrator

Penggunaan Data

Data yang kami kumpulkan digunakan untuk:

  • Menyediakan fitur manajemen tugas dan jadwal kuliah
  • Mengidentifikasi pengguna dalam sistem kami
  • Menyediakan pengalaman pengguna yang personal
  • Memungkinkan sinkronisasi data antar perangkat
  • Mengembangkan dan meningkatkan fitur aplikasi

Penyimpanan Data

Data Anda disimpan secara lokal di perangkat Anda dan disinkronkan secara aman ke server kami. Kami:

  • Menyimpan data hanya selama Anda menggunakan layanan
  • Menyediakan opsi untuk ekspor data pribadi Anda
  • Menghapus data Anda secara permanen jika akun dihapus

Keamanan

Kami menerapkan berbagai langkah keamanan untuk melindungi data Anda:

  • Enkripsi data saat transmisi dan penyimpanan
  • Sistem otentikasi berlapis dengan token
  • Validasi input untuk mencegah serangan
  • Pemantauan aktivitas sistem secara berkala

Hak Pengguna

Anda memiliki hak sebagai pengguna untuk:

  • Mengakses informasi yang telah Anda berikan
  • Memperbarui informasi pribadi Anda
  • Menghapus akun dan data pribadi Anda
  • Meminta penjelasan tentang penggunaan data Anda
