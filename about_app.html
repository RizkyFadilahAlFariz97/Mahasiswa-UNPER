Tentang Aplikasi

Sistem Manajemen Tugas & Jadwal Kuliah Mahasiswa UNPER

Apa Itu Aplikasi Ini?

Aplikasi Mahasiswa UNPER adalah sistem manajemen tugas dan jadwal kuliah yang dirancang khusus untuk membantu mahasiswa dalam mengatur aktivitas akademik mereka secara efektif dan terorganisir.

Tujuan Aplikasi

Aplikasi ini dibuat untuk membantu mahasiswa UNPER dalam:

  • Mengelola tugas dan deadline secara terorganisir
  • Memonitor jadwal kuliah mingguan
  • Mengatur prioritas tugas dengan jelas
  • Meningkatkan produktivitas akademik
  • Mengurangi lupa terhadap deadline tugas

Fitur Utama

  • Pembuatan dan manajemen tugas dengan berbagai prioritas
  • Penjadwalan kuliah mingguan
  • Kalender visual untuk melihat semua deadline
  • Fitur lampiran untuk dokumen atau file penting
  • Manajemen profil pengguna
  • Tema gelap untuk kenyamanan penggunaan
  • Filter dan pencarian tugas

Target Pengguna

Aplikasi ini ditujukan untuk seluruh mahasiswa Universitas Perjuangan (UNPER) yang ingin mengatur tugas kuliah dan jadwal kuliah mereka secara efektif dan terstruktur.

Keamanan Data

Kami menjamin keamanan data pribadi pengguna. Semua data disimpan secara lokal di perangkat pengguna dan tidak dikirim ke server pihak ketiga. Data hanya tersedia saat aplikasi digunakan di perangkat yang sama.

Desain Responsif

Aplikasi ini dirancang untuk dapat digunakan dengan baik di berbagai perangkat, mulai dari smartphone hingga tablet, memastikan kenyamanan dalam penggunaan sehari-hari.

User Experience

Dengan antarmuka yang intuitif dan ramah pengguna, aplikasi ini dirancang untuk memberikan pengalaman pengguna yang optimal dengan navigasi yang mudah dan tampilan yang bersih.

Teknologi yang Digunakan

Aplikasi ini dibangun dengan teknologi web modern:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js dengan Express.js
  • Database: SQLite untuk penyimpanan lokal
  • Desain: Responsive dan mobile-first approach
  • Framework: CSS custom dengan Poppins font

Manfaat Akademik

Dengan menggunakan aplikasi ini, diharapkan mahasiswa dapat:

  • Meningkatkan kedisiplinan dalam menyelesaikan tugas
  • Mengurangi kebingungan terhadap jadwal kuliah
  • Lebih siap dan terorganisir sebelum perkuliahan
  • Mengelola waktu secara lebih efektif
  • Meningkatkan kualitas hasil belajar
